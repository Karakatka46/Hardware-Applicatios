## Проект "Моргание светодиодом"
Данный проект написан с целью освоения принципов работы с микроконтроллерами STM32 и принципами объектно-ориентированного программирования.

Выбран подход bare-metal, в связи с чем используются дополнительные файлы:
- Линковщик **linker.ld**
- Ассемблерный файл запуска микроконтроллера **startup.s**
