# Hardware-Applicatios

## О проекте

Этот проект предназначен для обучения работе с C++ на микроконтроллерах STM32 (Nucleo), а также для освоения основ объектно-ориентированного программирования (ООП) на примере мигающего светодиода.  
Проект включает:

- Классы и структуры для работы с периферией.
- Примеры мигания светодиода на Nucleo в ООП стиле.
- Интеграцию с Linux через UART для отладки.
- Подготовку к задачам алгоритмов и структур данных на C++.
